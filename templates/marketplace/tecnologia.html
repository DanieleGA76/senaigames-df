Conceito e Tecnologias

Uma jornada através das ferramentas que impulsionam nosso projeto.

O Conceito Inicial

O projeto nasceu da ideia de criar uma experiência interativa que combinasse a potência da visão computacional com a análise de dados e a criação de interfaces lúdicas. O objetivo principal é desenvolver uma aplicação capaz de ler gestos e expressões, processá-los e traduzi-los em interações dinâmicas em tempo real. Isso abre portas para simulações educacionais, ferramentas de análise de comportamento e até mesmo jogos que são controlados apenas com o movimento do corpo.

Tecnologias Utilizadas

Python

Linguagem de programação fundamental, utilizada para a lógica principal do projeto, manipulação de dados e integração das bibliotecas.

MediaPipe

Framework de aprendizado de máquina de código aberto, essencial para a detecção de pontos de referência (landmarks) do corpo, como mãos e rosto, permitindo a interação por gestos.

Scikit-learn

Biblioteca de machine learning para Python, utilizada para análise de dados, classificação e treinamento de modelos baseados nas informações coletadas.

Pandas

Ferramenta poderosa para manipulação e análise de dados. Usado para organizar e limpar os dados capturados antes de serem processados por outros modelos.

Pygame

Biblioteca de desenvolvimento de jogos em Python. Fornece uma interface gráfica para visualizar os resultados da análise de gestos e criar a experiência interativa.
